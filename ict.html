Lesson 2: Where to Place Scripts

JavaScript can be placed in various locations within your HTML document. The placement affects when the script executes.

Script Placement Options:

  • In : Pre-loaded, executed before page events
  • In : Executed when page loads
  • External .js file: Linked with src attribute

External JavaScript:

Comments in JavaScript:

  • Multi-line: /* comment */

Syntax Rules:

Lesson 4: Events, Popup Boxes & Variables

Events:

Events are triggered by user actions. Common event attributes:

  • onclick, ondblclick
  • onload, onunload
  • onmouseover, onmouseout
  • onkeydown, onkeypress, onkeyup
  • onfocus, onblur
  • onsubmit, onreset

Popup Boxes:

  • Alert: alert("message")
  • Prompt: prompt("message","default")
  • Confirm: confirm("message")

Variables:

Temporary storage for values. Can hold numbers, strings, Boolean values.

Operators:

  • Arithmetic: +, -, *, /, %, ++, --
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !

Conditional Statements:

  • if
  • if-else
  • if-else if-else
  • switch
