Lesson 1: Introduction to CSS

CSS stands for Cascading Style Sheets. It defines how HTML elements are displayed across different media. CSS separates content from design, making websites easier to maintain and more visually appealing.

  • Controls layout across multiple pages
  • Stored in external .css files
  • Improves accessibility and responsiveness
  • Supports layered webpage structure: Content (HTML), Presentation (CSS), Behavior (JavaScript)
