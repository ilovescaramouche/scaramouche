Lesson 2: Structure of CSS

CSS is made up of selectors and declarations. A declaration includes a property and its value, which define how an HTML element should appear.

  • Selector: Targets the HTML element to style (e.g., h1)
  • Property: The style attribute to modify (e.g., color)
  • Value: The setting applied to the property (e.g., blue)
  • Declaration: A single instruction like color: blue;
  • Declaration block: A group of declarations inside curly braces

There are three types of CSS:

  • External: Linked via <link> tag and stored in a .css file
  • Embedded: Placed inside <style> tags in the HTML head
  • Inline: Written directly in the HTML tag using the style attribute

Each type has its own use case, and they follow a priority order: Inline > Embedded > External.
