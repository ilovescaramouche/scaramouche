Lesson 6: Dimensions and Pseudo-Classes

CSS can control visibility, width, and height of elements. Pseudo-classes like :hover and :active add interactivity. Below is a demo showing both.

Sample Code

<style>
  .hidden {
    visibility: hidden;
  }
  .box {
    width: 300px;
    height: 100px;
    background-color: lightblue;
    padding: 10px;
  }
  a:link { color: red; }
  a:visited { color: black; }
  a:hover { color: green; }
  a:active { color: blue; }
</style>

<div class="box">
  This box is visible.
  <span class="hidden">This span is hidden.</span>
</div>

<a href="#">Hover over me</a>

Output Preview

This box is visible. This span is hidden.
