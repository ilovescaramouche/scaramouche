Lesson 3: Display Property and Layout

The display property determines how HTML elements are arranged. Block elements stack vertically, while inline elements flow horizontally. Below is a live-style demo:

Sample Code

<style>
  p.inline-demo {
    display: inline;
    background-color: lightblue;
    padding: 5px;
    margin: 5px;
  }
</style>

<h1>Display Property Demo</h1>
<p class="inline-demo">Line 1</p>
<p class="inline-demo">Line 2</p>
<p class="inline-demo">Line 3</p>
