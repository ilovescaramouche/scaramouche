Lesson 4: CSS Classes and Selectors

CSS classes allow you to apply styles to specific elements using reusable names. You can target any HTML tag with a class, or be more specific by combining the tag and class name.

Sample Code

<style>
  p.note {
    background-color: lightyellow;
    border-left: 4px solid orange;
    padding: 10px;
    font-style: italic;
  }
</style>

<p>This paragraph has no class.</p>
<p class="note">This paragraph uses the "note" class.</p>
